
# BetGameKotlinApi - Spring boot + Kotlin + Docker + Docker Compose

<div align="center">
An Api that provides endpoints to register, bet, and retrieve bets.
</div>

# Start

## Steps
1 - Clone the repo and run the Spring boot + kotlin application

2 - Install Docker desktop app on your machine

3 - Open the project in your IDE, open the terminal

4 - Run this command below to init the application services (Postgresql Database)

`docker-compose up`

This command will create a new container into Docker environment and start up a new postgresql image.
